['Project Overview](#project-overview)\n- [Features](#features)\n- [Prerequisites](#prerequisites)\n- [Installation](#installation)\n- [Data](#data)\n- [Model](#model)\n- [Training](#training)\n- [Evaluation](#evaluation)\n- [Inference & Visualization](#inference--visualization)\n- [Configuration](#configuration)\n- [Contributing](#contributing)\n- [License](#license)\n\n## Project Overview\nThis project provides an end-to-end deployment of a machine learning model that classifies Iris flower species. The model is trained on the classic Iris dataset and deployed as a Streamlit web application that not only predicts the species but also visualizes prediction probabilities.\n\n## Features\n- Accurate species classification using a trained scikit-learn model\n- Real-time probability visualization with Plotly\n- Interactive web interface built with Streamlit\n- REST API endpoint for batch predictions\n- Dockerized deployment for consistent environments\n\n## Prerequisites\n- Python 3.10 or higher\n- pip package manager\n- Docker (for containerized deployment)\n- git\n\n## Installation\n1. Clone the repository:\n   ```bash\n   git clone https://github.com/your-org/iris-model.git\n   cd iris-model\n   ```\n2. Create a virtual environment:\n   ```bash\n   python -m venv venv\n   source venv/bin/activate  # On Windows use `venvScriptsactivate`\n   ```\n3. Install dependencies:\n   ```bash\n   pip install -r requirements.txt\n   ```\n\n## Data\nThe model is trained on the [Iris dataset](https://archive.ics.uci.edu/ml/datasets/iris) which contains 150 labeled instances of three Iris species. The dataset is included in the `data/` directory.\n\n- `data/iris.csv`: Raw dataset (sepal length, sepal width, petal length, petal width, species)\n- `data/processed/`: Processed features (optional)\n\n## Model\nThe classifier is a Support Vector Machine (SVM) with an RBF kernel, trained on the full feature set.\n- Model file: `models/iris_svm.joblib`\n- Training script: `train.py`\n- Model architecture details are logged in `models/model_metadata.json`\n\n## Training\nTo retrain the model:\n```bash\npython train.py\n```\nThe training script will:\n1. Load and preprocess the data\n2. Split into train/test sets\n3. Train the SVM classifier\n4. Evaluate performance (accuracy, classification report)\n5. Save the trained model and metadata\n\n## Evaluation\nModel performance is evaluated on the hold-out test set:\n- Accuracy: ~96%\n- Precision, Recall, F1-score per class\n- Confusion matrix\n\nEvaluation results are stored in `reports/evaluation.json`.\n\n## Inference & Visualization\nRun the Streamlit app to interact with the model:\n```bash\nstreamlit run app.py\n```\nThe app provides:\n- A form to input measurements (sepal/petal dimensions)\n- Real-time species prediction\n- Probability distribution visualization using Plotly bar chart\n\n## Configuration\n- `config.yaml`: Application configuration (e.g., model path, port numbers)\n- Environment variables can override settings\n\n## Contributing\n1.

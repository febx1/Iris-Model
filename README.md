Processed features are in `data/processed/`.\n\n> 📁 **File Structure**\n> - `data/iris.csv` – Raw features & labels  \n> - `data/processed/` – Scaled/encoded features (optional)  \n\n---\n\n## 🤖 Model\n- **Model file:** `models/iris_svm.joblib`  \n- **Metadata:** `models/model_metadata.json` (training hyperparameters, metrics)  \n- **Training script:** `train.py`  \n\n---\n\n## 🎓 Training\n```bash\npython train.py\n```\nThe script performs:\n1. Data loading & preprocessing  \n2. Train/test split (80/20)  \n3. SVM model fitting (RBF kernel)  \n4. Metrics computation (accuracy, precision, recall, F1, confusion matrix)  \n5. Persistence of model & metadata  \n\n---\n\n## 📈 Evaluation\nEvaluation results are stored in `reports/evaluation.json` with:\n\n- Overall accuracy (~96%)  \n- Per‑class precision, recall, F1  \n- Confusion matrix visualization  \n\n---\n\n## 🖥️ Inference & Visualization\nRun the interactive Streamlit app:\n\n```bash\nstreamlit run app.py\n```\n\nFeatures:\n- Input form (sepal/petal dimensions)  \n- Real‑time species prediction  \n- **Probability bar chart** (Plotly) showing class distribution  \n\n---\n\n## ⚙️ Configuration\n- `config.yaml` – Default config (model path, default port, etc.)  \n- Environment variables (`MODEL_PATH`, `PORT`, …) override config values  \n\nSample snippet (`config.yaml`):\n\n```yaml\nmodel_path: "models/iris_svm.joblib', 'app_port: 8501\ndebug: false\n```\n\n---\n\n## 🤝 Contributing\n1.
